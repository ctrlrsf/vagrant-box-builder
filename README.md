## Sample usage

```
$ ./build-box.sh xenial-0.0.3.box

==> default: Setting the name of the VM: vagrant-box-builder_default_1467985240377_50307
...
...
Running: vagrant package --base vagrant-box-builder_default_1467985240377_50307 --output xenial-0.0.3.box
==> vagrant-box-builder_default_1467985240377_50307: Attempting graceful shutdown of VM...
==> vagrant-box-builder_default_1467985240377_50307: Clearing any previously set forwarded ports...
==> vagrant-box-builder_default_1467985240377_50307: Exporting VM...
==> vagrant-box-builder_default_1467985240377_50307: Compressing package to: xenial-0.0.3.box
```
